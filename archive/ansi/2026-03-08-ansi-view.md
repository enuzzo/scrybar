# ANSI/BBS Art View — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Aggiungere una quinta view (UI_PAGE_ANSI) che mostra file .ANS in portrait
mode (172×640) con scroll automatico, font IBM VGA 8×16, palette CGA 16 colori,
e rotazione automatica tra i file dopo lo scroll completo.

**Architecture:** La view usa un buffer PSRAM (172 × totalHeight × 2 byte) in cui
viene renderizzato l'intero file ANSI una volta al caricamento. Il display riceve
il viewport direttamente (nessun tile-transpose) tramite `esp_lcd_panel_draw_bitmap`.
I file .ANS sono embedded come array C (`xxd -i`). LVGL resta attivo per le altre
view; la ANSI view bypassa LVGL per il rendering dei pixel, esattamente come
`updateDisplayClock()` bypassa LVGL per il canvas del clock.

**Tech Stack:** ESP-IDF `esp_lcd_panel_*`, PSRAM `heap_caps_malloc`, IBM VGA 8×16
font (array C 4096 byte, public domain), ANSI CSI color parser, SAUCE record parser.

**Display physicals:**
- `DB_NATIVE_W = 172`, `DB_NATIVE_H = 640` — queste sono le dimensioni del pannello
- La pipeline attuale: `g_canvasBuf` landscape 640×172 → `dispRotateChunk()` transpose
  software → `esp_lcd_panel_draw_bitmap(g_panel, ...)` in portrait 172×640
- Per ANSI: bypassiamo il transpose. Rendiamo direttamente in portrait 172×N,
  mandiamo con `esp_lcd_panel_draw_bitmap`. Serve cambiare `mirror` del pannello.
- Mirror attuale (init): `esp_lcd_panel_mirror(g_panel, true, true)` (DISPLAY_FLIP_180)
- Mirror per ANSI portrait (USB-C in alto): da calibrare in Task 3.
  Default nel piano: `mirror(false, false)`. Se l'immagine è speculare o capovolta,
  testare le 3 alternative: `(true,false)`, `(false,true)`, `(true,true)`.

**Orientamento fisico ANSI view:**
- Device tenuto in portrait con USB-C in ALTO
- Stabile su base piatta, cavo sale verso l'alto
- Contenuto scorre dall'alto verso il basso

```cpp
// x,y = posizione pixel top-left del carattere nel buffer.
// fontW, fontH = dimensione del glyph (default 8×16, può essere scalato).
static void drawAnsiChar(uint16_t *buf, int bufW, int x, int y,
                         uint8_t ch, uint8_t fgIdx, uint8_t bgIdx,
                         int fontW = 8, int fontH = 16) {
  const uint16_t fg = kAnsiPalette[fgIdx & 0x0F];
  const uint16_t bg = kAnsiPalette[bgIdx & 0x0F];
  for (int row = 0; row < fontH; ++row) {
    // Campiona la riga dal font 8×16 (anche se fontH != 16, scala)
    const int srcRow = (row * 16) / fontH;
    const uint8_t bits = kIbmVga8x16[(int)ch * 16 + srcRow];
    uint16_t *line = buf + (y + row) * bufW + x;
    for (int col = 0; col < fontW; ++col) {
      const int srcCol = (col * 8) / fontW;
      line[col] = (bits & (0x80 >> srcCol)) ? fg : bg;
    }
  }
}

#endif // TEST_DISPLAY
// ── fine ANSI font ────────────────────────────────────────────────────────────
```

## Task 2 — ANSI parser (CSI colors + SAUCE)

**Files:**
- Crea: `src/ansi/ansi_parser.h`

**Cos'è il formato .ANS:**
- Testo CP437 con escape sequences ANSI (CSI = `\x1B[`)
- Fine del contenuto display: `\x1A` (SUB/Ctrl-Z), seguito da record SAUCE
- SAUCE = "SAUCE00" + 128 byte di metadata (title 35b, author 20b, group 20b,
  year 4b, twidth 2b, theight 2b, ...)
- Escape sequences rilevanti:
  - `ESC[<n>m` — Select Graphic Rendition (colori)
    - 0 = reset (fg=7, bg=0)
    - 1 = bold (rende i colori "bright", aggiunge 8 all'indice fg)
    - 5 = blink (ignorare — trattare come bold)
    - 30-37 = fg colors (black=30, red=31, green=32, yellow=33, blue=34,
                          magenta=35, cyan=36, white=37)
    - 40-47 = bg colors
    - 90-97 = bright fg (= 30-37 ma con bright=true)
    - 100-107 = bright bg
  - `ESC[<row>;<col>H` — cursor position (1-based)
  - `ESC[<n>A/B/C/D` — cursor up/down/right/left (n steps)
  - `\r` = cursor to col 0; `\n` = cursor down 1 row (non reset col)
  - `\r\n` = newline completo

**Step 1: Crea `src/ansi/ansi_parser.h`**

```cpp
#pragma once
#include <stdint.h>
#include <string.h>

// Metadata estratti dal record SAUCE (opzionale, tutti i campi sono NUL-padded)
struct SauceRecord {
  char title[36];   // 35 chars + NUL
  char author[21];  // 20 chars + NUL
  char group[21];   // 20 chars + NUL
  char year[5];     // 4 chars + NUL
  uint16_t width;   // colonne ANSI (tipicamente 80)
  uint16_t height;  // righe ANSI
  bool valid;
};

// Estrae il record SAUCE dalla fine del file .ANS.
// Restituisce true se trovato. data/len = intero file.
static bool parseSauceRecord(const uint8_t *data, size_t len, SauceRecord &out) {
  out = {};
  // SAUCE record = 128 byte, preceduto da \x1A (può esserci prima CommentBlock)
  // Cerca "SAUCE00" negli ultimi 128 byte del file
  if (len < 128) return false;
  const uint8_t *rec = data + len - 128;
  if (memcmp(rec, "SAUCE00", 7) != 0) return false;
  // TitleID   = bytes 5..39 (35 chars)
  // Author    = bytes 42..61 (20 chars)  -- offset from rec start
  // Group     = bytes 62..81
  // Date      = bytes 82..89 (8 chars CCYYMMDD, prendiamo prime 4 per l'anno)
  // TInfo1    = bytes 96..97 (uint16 LE) = width
  // TInfo2    = bytes 98..99 (uint16 LE) = height
  strncpy(out.title,  (const char*)(rec + 7),  35); out.title[35] = 0;
  strncpy(out.author, (const char*)(rec + 42), 20); out.author[20] = 0;
  strncpy(out.group,  (const char*)(rec + 62), 20); out.group[20] = 0;
  strncpy(out.year,   (const char*)(rec + 82), 4);  out.year[4] = 0;
  out.width  = (uint16_t)(rec[96] | (rec[97] << 8));
  out.height = (uint16_t)(rec[98] | (rec[99] << 8));
  // Trim spazi finali
  for (int i = 34; i >= 0 && out.title[i]  == ' '; --i) out.title[i]  = 0;
  for (int i = 19; i >= 0 && out.author[i] == ' '; --i) out.author[i] = 0;
  for (int i = 19; i >= 0 && out.group[i]  == ' '; --i) out.group[i]  = 0;
  out.valid = true;
  return true;
}

// Stato del parser durante il rendering
struct AnsiParserState {
  int col = 0;       // colonna corrente (0-based)
  int row = 0;       // riga corrente (0-based)
  uint8_t fg = 7;    // foreground color index (0-15)
  uint8_t bg = 0;    // background color index (0-7)
  bool bold = false; // bold/bright attivo

  void reset() { col=0; row=0; fg=7; bg=0; bold=false; }

  uint8_t fgResolved() const { return bold ? (fg | 8) : fg; }
};

// Legge un intero CSI dalla sequenza ESC[...m e ritorna il puntatore dopo la sequenza.
// buf = puntatore a '[' (dopo ESC). Modifica state.
// Ritorna puntatore al byte DOPO la lettera finale della sequenza.
static const uint8_t* parseAnsiCsi(const uint8_t *buf, size_t remaining,
                                    AnsiParserState &st) {
  // Raccoglie parametri numerici separati da ';'
  int params[8] = {0};
  int nparams = 0;
  size_t i = 0;
  // Leggi fino alla lettera terminatrice
  while (i < remaining && nparams < 8) {
    uint8_t c = buf[i++];
    if (c >= '0' && c <= '9') {
      params[nparams] = params[nparams] * 10 + (c - '0');
    } else if (c == ';') {
      ++nparams;
    } else {
      // Lettera terminatrice
      if (nparams < 8) ++nparams; // finalizza ultimo param
      if (c == 'm') {
        // SGR — colori
        if (nparams == 0 || (nparams == 1 && params[0] == 0)) {
          st.fg = 7; st.bg = 0; st.bold = false; // reset
        } else {
          for (int p = 0; p < nparams; ++p) {
            int v = params[p];
            if (v == 0)             { st.fg=7; st.bg=0; st.bold=false; }
            else if (v == 1)        { st.bold = true; }
            else if (v == 5)        { st.bold = true; } // blink → treat as bright
            else if (v >= 30 && v <= 37) { st.fg = (uint8_t)(v - 30); }
            else if (v >= 40 && v <= 47) { st.bg = (uint8_t)(v - 40); }
            else if (v >= 90 && v <= 97) { st.fg = (uint8_t)(v - 90 + 8); }
            else if (v >= 100 && v <= 107) { st.bg = (uint8_t)(v - 100 + 8); }
          }
        }
      } else if (c == 'H' || c == 'f') {
        // Cursor position: ESC[row;colH (1-based)
        st.row = (nparams >= 1 && params[0] > 0) ? params[0] - 1 : 0;
        st.col = (nparams >= 2 && params[1] > 0) ? params[1] - 1 : 0;
      } else if (c == 'A') { st.row -= (nparams >= 1 && params[0] > 0) ? params[0] : 1; }
      else if (c == 'B') { st.row += (nparams >= 1 && params[0] > 0) ? params[0] : 1; }
      else if (c == 'C') { st.col += (nparams >= 1 && params[0] > 0) ? params[0] : 1; }
      else if (c == 'D') { st.col -= (nparams >= 1 && params[0] > 0) ? params[0] : 1; }
      // Ignora: J (clear screen), K (clear line), s/u (save/restore)
      break;
    }
  }
  return buf + i;
}
```

**Funzione `ansiRenderFile()`** (aggiungi nella stessa sezione ANSI, dopo `drawAnsiChar`):

```cpp
#if TEST_DISPLAY

#include "src/ansi/ansi_parser.h"

// Renderizza un file .ANS in g_ansiBuf (PSRAM).
// data/len = contenuto raw del file. bufW = larghezza buffer (= DB_NATIVE_W = 172).
// Ritorna l'altezza totale in pixel, 0 se errore.
static int ansiRenderFile(const uint8_t *data, size_t len,
                          uint16_t *buf, int bufW, int maxH,
                          int fontW, int fontH, SauceRecord &sauce) {
  // Trova fine contenuto display (\x1A) e parsa SAUCE
  parseSauceRecord(data, len, sauce);
  size_t displayLen = len;
  for (size_t i = 0; i < len; ++i) {
    if (data[i] == 0x1A) { displayLen = i; break; }
  }

  // Cancella buffer (colore bg=0 = nero)
  memset(buf, 0, (size_t)bufW * maxH * sizeof(uint16_t));

  AnsiParserState st;
  const uint8_t *p   = data;
  const uint8_t *end = data + displayLen;

  while (p < end) {
    uint8_t c = *p++;

    if (c == 0x1B && p < end && *p == '[') {
      // CSI sequence
      ++p; // skip '['
      p = parseAnsiCsi(p, (size_t)(end - p), st);
      continue;
    }
    if (c == '\r') { st.col = 0; continue; }
    if (c == '\n') {
      // Newline: avanza riga, riempie resto della riga corrente con bg
      ++st.row;
      st.col = 0;
      continue;
    }
    if (c < 0x20) continue; // altri control chars: ignora

    // Carattere stampabile
    const int px = st.col * fontW;
    const int py = st.row * fontH;
    if (py + fontH <= maxH && px + fontW <= bufW) {
      drawAnsiChar(buf, bufW, px, py, c, st.fgResolved(), st.bg, fontW, fontH);
    }
    ++st.col;
    // Wrap colonna
    const int maxCols = bufW / fontW;
    if (st.col >= maxCols) { st.col = 0; ++st.row; }
  }

  // Altezza effettiva = ultima riga usata + 1, in pixel
  // (st.row dopo il parsing = ultima riga scritta)
  return min((st.row + 1) * fontH, maxH);
}

// Calcola fontW/fontH ottimali per far stare l'ANSI in DB_NATIVE_W (172px).
// Usa il campo width del SAUCE se disponibile, altrimenti 80.
static void ansiCalcFont(const SauceRecord &sauce, int &fontW, int &fontH) {
  const int cols = (sauce.valid && sauce.width > 0 && sauce.width <= 160)
                   ? sauce.width : 80;
  // Trova il font size che fa stare 'cols' colonne in 172px
  fontW = DB_NATIVE_W / cols;
  if (fontW < 1) fontW = 1;
  fontH = fontW * 2; // aspect ratio 1:2
  // Minimo: font 4×8
  if (fontW < 4) { fontW = 4; fontH = 8; }
  // Massimo: font 8×16 (qualità IBM VGA originale)
  if (fontW > 8) { fontW = 8; fontH = 16; }
}
```

```cpp
// Manda il viewport corrente (172 × DB_NATIVE_H) al display direttamente.
// g_ansiScrollY = prima riga pixel nel buffer.
static void ansiFlushViewport() {
  if (!g_ansiBuf || g_ansiTotalH == 0) return;
  const int scrollY = min(g_ansiScrollY, max(0, g_ansiTotalH - DB_NATIVE_H));
  const uint16_t *src = g_ansiBuf + scrollY * DB_NATIVE_W;
  // Il pannello accetta tutto il frame in una volta (CASET/RASET non funzionanti)
  // Usiamo il DMA buffer come staging area a 64-row chunks
  const int chunks = DB_NATIVE_H / DB_CHUNK_ROWS; // = 640/64 = 10
  for (int c = 0; c < chunks; ++c) {
    const uint16_t *chunk = src + c * DB_CHUNK_ROWS * DB_NATIVE_W;
    esp_lcd_panel_draw_bitmap(g_panel, 0, c * DB_CHUNK_ROWS,
                              DB_NATIVE_W, (c + 1) * DB_CHUNK_ROWS,
                              (void*)chunk);
  }
  // Se l'altezza del buffer è minore di DB_NATIVE_H (file corto), neriamo il resto
  const int rendered = g_ansiTotalH - scrollY;
  if (rendered < DB_NATIVE_H) {
    // fill nero per il resto — usa g_dmaBuf come scratch
    memset(g_dmaBuf, 0, DB_NATIVE_W * DB_CHUNK_ROWS * sizeof(uint16_t));
    const int firstBlankChunk = rendered / DB_CHUNK_ROWS;
    for (int c = firstBlankChunk; c < chunks; ++c) {
      esp_lcd_panel_draw_bitmap(g_panel, 0, c * DB_CHUNK_ROWS,
                                DB_NATIVE_W, (c + 1) * DB_CHUNK_ROWS,
                                g_dmaBuf);
    }
  }
}

#endif
```

```cpp
  g_ansiBuf = (uint16_t*)heap_caps_malloc(needed, MALLOC_CAP_SPIRAM);

    g_ansiTotalH = 0;

                f.filename, fontW, fontH, DB_NATIVE_W / fontW, g_ansiTotalH,
                (unsigned)needed);
```
